Gift Bag Tags

Designed for a client, these tags add a personalized touch to gifts.

Process

Selected colors, motifs, and typefaces to create charming, cohesive tags for various occasions.

Tools Used

  • Adobe Illustrator
  • Adobe Photoshop

Accessibility Statement

This project includes basic accessibility features such as semantic HTML structure and descriptive alt text for images. However, the work is primarily visual in nature, relying on decorative design elements that may not fully translate for users using screen readers. Future improvements could include more detailed text descriptions of design variations, improved color contrast verification, and additional accessibility testing to ensure broader usability.
