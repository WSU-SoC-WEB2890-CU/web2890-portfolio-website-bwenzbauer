Waterslide Model Decals

Client project designing decals for a waterslide model, ensuring realism and visual appeal.

Process

Designed detailed graphics, aligned to scale, and tested readability from various distances.

Tools Used

  • Adobe Illustrator

Accessibility Statement

This project includes basic accessibility considerations such as semantic HTML structure and descriptive alt text for images. However, because the work focuses on detailed visual decals, some elements may not be fully perceivable to users relying on screen readers or other assistive technologies. Future improvements could include more detailed written descriptions of decal designs, improved contrast verification for small-scale elements, and additional accessibility testing to ensure clarity across different viewing conditions.
