Accessibility Statement

This project incorporates accessibility fundamentals such as semantic HTML structure, responsive design principles, and descriptive alt text for images. The layout is built using Bootstrap to support mobile responsiveness and improved usability across devices. However, some accessibility limitations may remain, including potential reliance on visual layout and color contrast. Future improvements could include enhanced keyboard navigation support, ARIA attributes where needed, and more comprehensive screen reader testing.
