Accessibility Statement

This project includes basic accessibility considerations such as semantic HTML structure and descriptive alt text for images. While the design is visually engaging and uses soft colors and illustrations, some users with visual impairments may experience limitations due to reliance on color and imagery. Future improvements could include stronger color contrast, expanded text alternatives for visual elements, and additional support for screen readers and keyboard navigation.
