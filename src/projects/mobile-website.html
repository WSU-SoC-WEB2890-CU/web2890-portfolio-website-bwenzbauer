Mobile Website

Redesign of a website for mobile devices as a Davis Technical College project.

Process

Focused on responsive design, UX improvements, and mobile-first layout principles.

Tools Used

  • Adobe XD
  • Figma

Accessibility Statement

This project incorporates accessibility-aware design practices such as responsive layout principles and structured semantic HTML. The mobile-first approach helps improve usability across different devices and screen sizes. However, additional accessibility enhancements could still be made, such as more explicit ARIA labeling, improved focus states for keyboard navigation, and further screen reader testing to ensure full inclusivity.
