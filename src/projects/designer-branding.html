Designer Branding

Davis Technical College SkillsUSA project focused on personal branding for a designer portfolio.

Process

Created cohesive visual identity including logo, typography, and brand colors.

Tools Used

  • Adobe Illustrator
  • Adobe Photoshop

Accessibility Statement

This project includes foundational accessibility features such as semantic HTML structure and descriptive alt text for images. The design focuses heavily on visual branding elements like typography, color, and layout consistency, which may not fully translate for users with visual impairments. Future improvements could include increased color contrast checks, expanded text-based descriptions of visual branding elements, and further testing with screen readers to ensure a more inclusive experience.
